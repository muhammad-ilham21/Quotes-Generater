# 🎯 JavaScript Quote Generator

A simple and elegant **Random Quote Generator** built with HTML, CSS, and JavaScript.  
Click the button to display a new inspirational quote each time.

---

## ✨ Features
- Displays random quotes from a predefined list
- Clean and modern UI with background image
- Responsive design for desktop and mobile
- Easy to extend with more quotes
